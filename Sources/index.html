Parsing DICOM using WebAssembly

  • This demonstration is entirely run client-side by the Web browser using the WebAssembly technology.
  • The DICOM parsing is fully implemented in C++ using the industrial-grade DCMTK toolkit.
  • The JavaScript code contains no intelligence about DICOM. It just acts as a wrapper to upload the DICOM file to the C++ code.
  • Make sure that your browser supports WebAssembly. Mozilla Firefox 52, Google Chrome 57 and Microsoft Edge 15 are known to work.
  • This sample is provided courtesy of Osimis.
